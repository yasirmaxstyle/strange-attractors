## About

This project explores the chaotic beauty of strange attractors—complex mathematical systems where points evolve dynamically in 3D space. The visualizations are rendered in real-time using WebGL, leveraging the power of **Three.js** and **React Three Fiber** for declarative 3D graphics in a modern React environment.

## Features

- **Real-time 3D Rendering**: High-performance WebGL graphics powered by Three.js.
- **Interactive Controls**: Tweak parameters on the fly using `leva` to see how the attractors react to variable changes.
- **Post-Processing**: Beautiful visual effects utilizing `@react-three/postprocessing`.
- **Modern Stack**: Built with React 19, TypeScript, and Vite for a fast and robust development experience.

## Technology Stack

- **[React](https://react.dev/)** - UI library
- **[Three.js](https://threejs.org/)** - 3D JavaScript library
- **[React Three Fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction)** - React renderer for Three.js
- **[@react-three/drei](https://github.com/pmndrs/drei)** - Useful helpers and abstractions for React Three Fiber
- **[Vite](https://vitejs.dev/)** - Next Generation Frontend Tooling
- **[TypeScript](https://www.typescriptlang.org/)** - Strongly typed programming language

## Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yasirmaxstyle/strange-attractors.git
   cd strange-attractors
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

## Scripts

- `npm run dev`: Starts the local development server with Hot Module Replacement (HMR).
- `npm run build`: Compiles TypeScript and builds the app for production.
- `npm run preview`: Locally previews the production build.
- `npm run deploy`: Builds and deploys the application directly to GitHub Pages.
